#Tricking-Stat-Tracker
# -Read-ME-IMPORTANT
Tricking is an atheletic sport that mixes elements from gymnastics, martial arts, breakdancing.It's believed that tricking comes from tribal ritual practices from Africa/ East Asia. Tricking is an upcoming sport that has experienced exponential growth for the last thirty years. Throughout the careers of elite-level trickers they gain skills and experiences and at the moment the only way to track these achievements is through video evidence or memory.

The goal of this Tricking-Stat-Tracker is to give Trickers a place where they can store and readily find their achievements and goals. As Tricking is a relatively new sport, it is in dire need of more infrastructure.

>-START-
Tricking-Stat-Tracker recognized user's off a first and last name basis.
After the system recognizes user it will take input on if you want to store an achievement, look at current achievements, or set a goal.

>-STORING DATA-
The data that is being stored with the user will be in two groups, "Gatherings Attended" and "Tricks Landed"

>-GATHERINGS ATTENDED-
If Gatherings Attended is chosen input, system will additional input of; "What gathering did you attend?", "What year did you attend this gathering", "How would you rate the gathering out of 5 stars?", "Leave a review of this gathering?". After the last question the system will tell the user "Your Gathering is now logged!" then bring the user to the store,look, or set goal page.

>-TRICKS LANDED-
If Tricks Landed is the chosen input, the system will ask the user for input of; "What trick do you want to log?", "When did you land this trick?" , "How would you rate your mastery of this trick now out of 5 stars?" after getting input system tells the user "Thank you, Your trick is logged!" then brings user back to store,look, or set goal page.

>-DISPLAYING ACHIEVEMENTS-
If user chooses input of displaying data, the system whill check the user for any existing achievements if none exist system outputs "You have no achievements! Want to log some?".
If user does have existing data it is displayed with Gatherings attended with its own column and Tricks landed in its column. The ratings and review is included on the gatherng column.
And rating is included with the list of tricks landed.

>-SET A GOAL-
If user chooses set goal option, the system will ask the user for input of; "What type of goal do you want to set?", "Gathering or Trick?". If chosen option is Gathering, user input what gathering they want to attend and the date of the gathering. If chosen option is Trick, system ask user for input of; "What trick are you trying to land?", "How long do you think it will take to land?"
>After setting goal system outputs "Your goals are now set, keep on tricking!"
